# Cloyva

**Cloyva** is an early-stage mobile app concept for a social fashion marketplace where people can **borrow, buy, swap, and style clothes** from real closets around them.

The goal is to build a modern, international fashion app that combines the best parts of a secondhand marketplace, a digital wardrobe, social style discovery, and AI-powered outfit inspiration.

> **Borrow. Buy. Swap. Style.**

---

## Vision

Cloyva helps people discover clothes and outfits from nearby users, try new styles without always buying something new, and turn their own wardrobe into a social fashion profile.

Instead of being only a resale app, Cloyva is designed around four core actions:

- **Borrow** clothes for events, weekends, photoshoots, trips, or everyday outfits
- **Buy** secondhand pieces directly from other users
- **Swap** clothes with people who have a similar style
- **Style** outfits with help from AI suggestions and social inspiration

---

## Core Idea

Most people have clothes they do not wear often, while others are looking for unique, affordable, and stylish pieces. Cloyva connects these two sides through a simple mobile experience.

A user can upload items from their closet and choose whether each item is:

- Private
- Visible for inspiration
- Available to borrow
- Available for sale
- Available for swap
- Available for both borrowing and buying

---

## Main Features

### 1. Style Feed

A visual feed where users can discover outfits, clothing items, and closets from people nearby or around the world.

Example actions:

- Save an item
- Borrow an item
- Buy an item
- Offer a swap
- Follow a closet

### 2. Digital Closet

Each user has a personal closet profile where they can upload and manage clothing items.

Each item can include:

- Photos
- Category
- Size
- Condition
- Style tags
- Borrow price
- Buy price
- Availability
- Location or shipping option

### 3. Borrow-to-Buy

Users can borrow an item first and later buy it if they like it.

Example:

- Borrow for €8
- Buy for €45
- If the user buys the item, the borrow price can be deducted from the final price

### 4. Style Match

Cloyva can help users find people with similar style, size, and fashion preferences.

Possible match signals:

- Style tags
- Clothing sizes
- Saved items
- Location
- Closet categories
- Outfit preferences

### 5. AI Stylist

AI can help users create outfits from their own closet and from items available nearby.

Example suggestions:

- “Pairs well with black jeans and white sneakers.”
- “Good for a casual weekend outfit.”
- “This jacket matches your streetwear style.”

### 6. Requests and Inbox

Users can manage borrow, buy, and swap requests from one simple inbox.

Request types:

- Borrow request
- Buy request
- Swap offer
- Pickup or delivery message
- Return confirmation

---

## MVP Scope

The first version should stay simple. The goal is to test whether people actually want to upload clothes, browse nearby closets, and send borrow, buy, or swap requests.

### MVP Features

- User profile
- Style and size preferences
- Add clothing item
- Feed of items
- Item detail page
- Closet/profile page
- Borrow, buy, and swap request buttons
- Simple inbox or request system

### Not Needed in the First Version

- Full AI outfit generation
- Automatic payments
- Shipping integration
- Complex verification system
- Advanced recommendation algorithm
- Large admin dashboard

These can be added later if the MVP proves that users like the concept.

## Trust and Safety Ideas

Because Cloyva involves borrowing and selling real items between people, trust is a key part of the product.

Possible trust features:

- Verified phone number
- Verified payment method
- User ratings
- Item condition photos
- Pickup photo
- Return photo
- Deposit system
- Late return tracking
- Report user button
